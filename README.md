#🔥 TITLE: Hand-Arm Vibration Modeling and Analysis (Spring-Mass-Damper System)#

### PROJECT SUMMARY
This project models vibration transmission in a grinder handle using a 2-DOF Spring-Mass-Damper system. It combines:

### 💡 Analytical derivation
- Frequency-domain analysis
- Python simulation
- Signal processing (FFT & PSD)

### ⚙️ METHODS
- Newton’s laws → equations of motion
- Matrix formulation (M, C, K)
- Frequency response analysis
- Transmissibility evaluation

### 📊 RESULTS
- Resonance peaks identified
- Frequency response validated
- Vibration amplification zones detected

### 🧠 SKILLS DEMONSTRATED
- Mechanical system modeling
- Python numerical simulation
- Signal processing (FFT, PSD)
- Engineering interpretation
